## 📰 News Aggregator

A full-stack News Aggregator platform that compiles and displays the latest news articles from multiple sources in a single interface.

## Features
🌍 Browse top headlines and country-specific news

💾 Persistent storage with MySQL

🎨 Responsive React frontend
🏗️ Tech Stack
## Backend
Java 21
Spring Boot 3.2.x (Web, JPA)
MySQL Database
Maven for build & dependency management

## Frontend

React + Vite

React Router DOM

Axios for API calls

TailwindCSS (if styling applied)

## 🚀 Getting Started
Prerequisites
Java 21
Node.js 18+
MySQL running locally 
Maven
Git

🔧 Setup
1️⃣ Clone the repo
git clone https://github.com/<your-username>/news-aggregator.git
cd news-aggregator

2️⃣ Backend Setup
cd spring_backend/app
mvn clean install
mvn spring-boot:run
Backend runs at: http://localhost:8080

3️⃣ Frontend Setup
cd react-frontend
npm install
npm run dev

Frontend runs at: http://localhost:5173
